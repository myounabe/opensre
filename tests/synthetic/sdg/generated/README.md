# Generated synthetic scenarios

Pipeline output goes here using the **same layout** as `tests/synthetic/rds_postgres/NNN-slug/`:

- `scenario.yml`
- `answer.yml`
- `alert.json`
- `cloudwatch_metrics.json`
- `rds_events.json`
- `performance_insights.json`

Subfolders are gitignored except this file. Set `paths.generated_scenarios_dir` in `../sdg_config.yml` if you want a different output root.
